# ASSERT Results Viewer

Quick local UI for browsing evaluation artifacts under `artifacts/results/`.

Canonical documentation lives in `docs/guides/local-viewer.md`.

## Quick start

```sh
npm install
npm run dev
```

Open `http://localhost:5174`.

## Common commands

```sh
npm run build
npm run preview
npm run check
```

## Notes

- Requires Node.js 18+.
- The viewer does not provide authentication; use your own proxy if needed.
- For artifacts, read-model expectations, and troubleshooting, see `docs/guides/local-viewer.md`.
