## Which one should I read first?

- **Have a framework agent (LangGraph, CrewAI, OpenAI Agents SDK, ...) ?** → [Travel Planner](/adaptive-eval/examples/travel-planner-otel/)
- **Just have a system prompt and a model, want to iterate?** → [Health Assistant](/adaptive-eval/examples/health-assistant-prompt-agent/)
- **Need to demonstrate eval → fix → re-eval?** → [Triage Agent](/adaptive-eval/examples/triage-agent-eval-fix/)
